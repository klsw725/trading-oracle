## 설계 결정

1. **가중치 저장 없음** — 매 실행 시 스냅샷에서 on-the-fly 계산. 별도 상태 파일 관리 불필요.
2. **Cold start**: 스냅샷 5개 미만이면 `None` → 기존 동등 가중치 (하위 호환 100%)
3. **만장일치 예외**: 5/5 전원 동일이면 가중치 무관하게 만장일치 유지
4. **가중 분류 임계값**: ratio >= 0.8 → 강한 합의, >= 0.6 → 약한 합의, 미만 → 분기
5. **최소 가중치 0.1**: 적중률 0%여도 완전히 무시하지 않음 (관점 다양성 보존)

## 검증

- `compute_perspective_weights()`: 스냅샷 1개 → `None` ✅
- `compute_consensus(weights=None)`: 기존 동등 가중치 동작 ✅
- `compute_consensus(weights={...})`: 가중 투표로 결과 변경 확인 ✅
- 만장일치: 가중치 무관 유지 ✅
- `performance.py report`: cold start 메시지 표시 ✅
- `daily.py --no-weights --help`: 플래그 동작 ✅
