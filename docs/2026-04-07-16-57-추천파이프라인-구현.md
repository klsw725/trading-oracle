## 변경 사항

- `scripts/recommend.py`
  - 기본 시장을 `KR`로 변경했다.
  - `--market` 도움말에 `KR=KOSPI+KOSDAQ`, `US=NASDAQ+NYSE`, `ALL=KR+US` 의미를 반영했다.
  - 터미널 출력에 universe 크기, market breakdown, selection constraints, 종목별 market/sector/selected_by 정보를 표시하도록 확장했다.

- `src/common.py`
  - `run_recommend()` 기본 시장을 `KR`로 변경했다.
  - widened universe + diversified selection 결과를 사용하는 `screen_recommendation_candidates()` 경로로 연결했다.
  - 결과 JSON에 `universe_size`, `universe_breakdown`, `selection_constraints` 메타데이터를 포함했다.
  - 추천 항목에 `market`, `sector`, `selected_by`를 포함했다.

- `src/screener/leading.py`
  - recommendation 전용 universe 구축 경로를 추가했다.
  - `KR`, `US`, `ALL` composite market semantics를 구현했다.
  - 시장별 기본 universe size(`KOSPI/KOSDAQ/NASDAQ/NYSE = 50`)를 적용했다.
  - 점수 우선 + 섹터 cap + 시장 균형 선호 + 단계적 완화 기반 greedy diversified selection을 추가했다.
  - 미국 listing의 `Industry` 컬럼을 섹터 분류 우선순위에 연결했다.

- `src/portfolio/correlation.py`
  - `classify_sector()`가 listing sector/industry 값을 우선 사용하도록 확장했다.

- `config.yaml`
  - `recommend.default_market`, `recommend.universe_size`, `recommend.diversification` 설정 섹션을 추가했다.

## 검증

- `uv run python -m py_compile src/screener/leading.py src/common.py scripts/recommend.py src/portfolio/correlation.py`
- `uv run scripts/recommend.py --market KR --top 6 --no-llm --json`
- `uv run scripts/recommend.py --market US --top 4 --no-llm --json`
- `uv run scripts/recommend.py --market ALL --top 4 --no-llm --json`
