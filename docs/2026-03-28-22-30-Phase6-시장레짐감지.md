## 설계 결정

1. **코드 기반 판정** — LLM 불필요, API 비용 0
2. **판정 기준**: 코스피 20일 수익률 + EMA(20) 위치 조합
   - bull: >3% AND 종가 > EMA(20)
   - bear: <-3% AND 종가 < EMA(20)
   - sideways: 그 외
3. **value/quant 관점 제외** — 시장 레짐과 직접 관련 없는 관점 (절대 밸류에이션, 종목 시그널)
4. **별도 파일 불필요** — 20줄 미만 함수, common.py에 인라인
