# 시그널 백테스트 구현

**날짜**: 2026-03-29
**프롬프트**: 과거 데이터 기반으로 체크해서 검증을 해보고 싶다 → 구현

## 변경 사항

### 신규 파일
- `scripts/backtest.py` — 과거 OHLCV 기반 기술 시그널 적중률 검증 스크립트

### 기능
- 과거 N일치 OHLCV를 일별로 슬라이싱하여 `compute_signals()` 실행
- 각 거래일의 BULLISH/BEARISH/NEUTRAL 판정 vs 실제 T+5일, T+20일 수익률 비교
- 적중 판정 기준은 `performance/tracker.py`와 동일:
  - BUY 5일: 수익률 > 0%
  - BUY 20일: 수익률 > 3%
  - SELL 5일: 수익률 < 0%
  - SELL 20일: 수익률 < -3%
  - HOLD 5일: |수익률| < 3%
  - HOLD 20일: |수익률| < 5%

### 출력 지표
- 판정별(BULLISH/BEARISH/NEUTRAL) × 윈도우별(5일/20일):
  - 적중률, 승률, 평균/중앙값/표준편차 수익률, 손익비(profit factor), 최고/최저
- 전략 요약 (BULL 매수 + BEAR 공매도): 누적 수익, 시그널당 수익, MDD, 샤프 근사치

### 사용법
```bash
uv run scripts/backtest.py 005930                   # 삼성전자 1년 백테스트
uv run scripts/backtest.py 005930 000660 035420      # 여러 종목
uv run scripts/backtest.py 005930 --days 500         # 기간 지정
uv run scripts/backtest.py 005930 --windows 5 10 20  # 평가 윈도우 지정
uv run scripts/backtest.py 005930 --json             # JSON 출력
```
