## 변경 사항

### `README.md`

- 추천 기능 설명을 기존 "1-step 종목 추천" 중심 표현에서 현재 추천 파이프라인 중심 설명으로 갱신했다.
- 빠른 시작 예시에 `KR` 기본 추천, `US`, `ALL` 추천 예시를 추가했다.
- `scripts/recommend.py` 설명을 `KR/US/ALL + diversified selection` 기준으로 갱신했다.
- 사용자 요청별 명령 매핑에 `전체 시장에서 뭐 살까?` 예시를 추가했다.
- `종목 추천 파이프라인` 섹션을 추가해 아래를 명시했다.
  - 기본 `--market = KR`
  - `KR = KOSPI + KOSDAQ`
  - `US = NASDAQ + NYSE`
  - `ALL = KR + US`
  - `--top`은 최종 분석 대상 수
  - `universe_size`, `universe_breakdown`, `selection_constraints`, `market`, `sector`, `selected_by`

### `SKILL.md`

- 스킬 설명을 한국 전용이 아닌 한국+미국 시장 기준으로 갱신했다.
- 워크플로우 예시에 `KR` 기본 추천, `US`, `ALL` 추천 예시를 추가했다.
- 추천 섹션 제목과 내용을 현재 Recommendation Pipeline 기준으로 갱신했다.
- 추천 섹션에 아래를 명시했다.
  - 기본 `--market = KR`
  - `KR / US / ALL` semantics
  - `--top` 의미 변경
  - widened universe → score → diversified selection → signal filter → multi-perspective 흐름
  - 추천 결과 메타데이터
- 사용자 요청 매핑에 `전체 시장에서 뭐 살까?` 명령을 추가했다.

## 검증

- `uv run scripts/recommend.py --json`
- `uv run scripts/recommend.py --market US --no-llm --json`
- `uv run scripts/recommend.py --market ALL --no-llm --json`

## 메모

- `--market US --json`, `--market ALL --json`의 전체 LLM 경로는 이 환경에서 실행 시간이 길어 툴 타임아웃에 걸릴 수 있다.
- 시장 semantics와 출력 메타데이터 검증은 `--no-llm` 경로로 확인했다.
