## 문제

v2의 적응형 가중치(Phase 5)는 전체 기간 적중률 기반의 단일 가중치 세트. 상승장에서 잘 맞추는 관점과 하락장에서 잘 맞추는 관점이 다른데, 이를 구분하지 않는다.

## 솔루션

Phase 14의 레짐별 적중률 행렬을 가중치로 변환. 현재 시장 레짐에 따라 해당 레짐 최적 가중치를 자동 선택.

## 마일스톤

### M1: 레짐별 가중치 계산
- [ ] `compute_regime_weights(regime)` 구현
- [ ] 레짐별 최소 표본 수 미달 시 v2 가중치 폴백

### M2: 파이프라인 연동
- [ ] `run_multi_perspective()`에서 레짐별 가중치 우선 사용
- [ ] 가중치 전환 시 로그 + JSON 출력에 레짐별 가중치 포함

### M3: 모니터링
- [ ] 성과 리포트에 "현재 적용 중인 가중치 세트" 표시
- [ ] 레짐 전환 시 경고 (가중치가 크게 변경될 때)
