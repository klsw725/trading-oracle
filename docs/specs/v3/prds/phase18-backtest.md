> **SPEC 참조**: [SPEC.md](../SPEC.md)

> **우선순위**: P1
> **선행 조건**: Phase 9 시그널 엔진(완료), Phase 17 환율 팩터(완료)

---

## 문제

17개 Phase를 구현했지만 시스템의 실제 수익성을 검증할 방법이 없다. 백테스트 없이는:
1. 시그널 앙상블의 과거 성과를 측정할 수 없음
2. Phase 17 환율 팩터의 실제 효과를 A/B 비교할 수 없음
3. v3 자가 학습(스냅샷 30개+)을 수개월 기다려야 함

## 솔루션

시그널 기반 백테스트 엔진. LLM 호출 없이 6-시그널 앙상블 + 환율 시그널로 매매 시뮬레이션.

## 설계 원칙

- **LLM 비용 0**: 시그널 레이어(코드)만 사용. LLM 관점은 백테스트 대상 아님.
- **현실적 시뮬레이션**: 슬리피지, 거래 수수료, 현금 비중 하한 반영.
- **비교 가능**: 환율 팩터 ON/OFF, 파라미터 변경 등 A/B 테스트 지원.
