# PRD: Phase 7 — 일간 변동 리포트 (Daily Delta Report)

> **SPEC 참조**: [SPEC.md §1-1 (매일 아침 투자 조언)](../SPEC.md#1-1-매일-아침-투자-조언-핵심-시나리오)
> **상태**: ✅ 완료 (M1~M2 구현)
> **우선순위**: P1 — Phase 4 (스냅샷) 완료 후 착수
> **선행 조건**: Phase 4 (추천 성과 추적) 완료

---

## 문제

사용자가 매일 `daily.py`를 실행하면 오늘의 분석 결과만 보인다. 어제 대비 무엇이 바뀌었는지(판정 전환, 관점 변동) 비교할 수단이 없어서, 변화가 있을 때 즉시 알아채기 어렵다.

## 솔루션

이전 스냅샷과 현재 분석 결과를 비교하여 합의 verdict 변경, 관점별 변동, 새 종목/제거 종목을 자동 표시.

---

## 마일스톤

### M1: Delta 계산 로직
- [x] `src/performance/tracker.py`에 `compute_delta()` 추가
- [x] 합의 verdict 변경 감지 + 관점별 변동 추적
- [x] 새 종목 / 제거 종목 감지
- [x] 이전 스냅샷 없으면 None 반환 (graceful)

**검증**: 동일 데이터→변동 없음, verdict 변경→변동 감지, 종목 추가/제거 감지 ✅

### M2: 출력 연동
- [x] `daily.py` 터미널에 변동 섹션 표시 (변동 없으면 "전일 대비 변동 없음")
- [x] `daily.py` JSON 출력에 `delta` 필드 포함
- [x] `main.py`에도 동일 적용

**검증**: daily.py 실행 시 delta 섹션 표시 확인 ✅
