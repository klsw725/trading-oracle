## 데이터 흐름
```
macro.py (다통화 시계열)
  → forex.py:detect_fx_regime (환율 레짐)
  → forex.py:compute_fx_signal (종목별 시그널)
  → perspectives/macro.py (프롬프트 주입)
  → sizer.py (포지션 사이징 ±15%)
```

## 비파괴 원칙
- 모든 환율 로직은 None 체크 후 스킵
- FX 데이터 수집 실패 → 레짐 없음, 조정 계수 1.0
- 기존 파이프라인은 환율 없이도 100% 동작
