# 추천 리스트 매수 가능 배지 표시

## 프롬프트

> 추천 리스트에서 “매수 가능 / 매수 불가”를 더 눈에 띄게 뱃지처럼 표시

## 변경 사항

### `scripts/recommend.py`

- 추천 카드마다 상단에 Rich 마크업 배지를 추가
  - 매수 가능: `[bold black on green] 매수 가능 [/bold black on green]`
  - 매수 불가: `[bold white on red] 매수 불가 [/bold white on red]`
- 카드 제목에도 동일 배지를 포함해 스캔할 때 바로 구분되도록 변경
- `action_plan.type == "buy"`면 초록 패널, 그 외(특히 `buy_blocked`)는 빨간 패널 스타일을 사용하도록 조정

## 결과

- 추천 리스트에서 각 종목이 현재 자금 기준으로 바로 매수 가능한지 한눈에 식별할 수 있다.
- 기존 매수 전략/매수 차단 상세 문구는 그대로 유지된다.

## 검증

```bash
uv run python -m py_compile scripts/recommend.py
```

- `py_compile` 통과
- `lsp_diagnostics`에는 이 파일의 기존 basedpyright 경고가 남아 있으나, 이번 변경과 직접 관련된 신규 구문 오류는 없다.
