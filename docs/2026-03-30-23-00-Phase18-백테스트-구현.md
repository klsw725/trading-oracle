## 신규 파일
- `src/backtest/__init__.py`
- `src/backtest/engine.py` — 시그널 기반 백테스트 엔진
- `src/backtest/metrics.py` — 성과 지표 (CAGR, MDD, 샤프, 승률)
- `scripts/backtest.py` — CLI (--period, --tickers, --compare, --json)
- `docs/specs/v3/prds/phase18-backtest.md` — PRD

- `docs/specs/v3/SPEC.md` — Phase 18 연결

## 사용법
```bash
# 6개월 백테스트
uv run scripts/backtest.py --period 6m --tickers 005930,000660,005380

# 환율 팩터 A/B 비교
uv run scripts/backtest.py --period 6m --tickers 005930,000660,005380 --compare

# JSON 출력
uv run scripts/backtest.py --period 6m --json
```
