# 일일 분석 출력에 Phase 17~19 통합

> 프롬프트: "D Next" → 출력 통합 추천 → "go"

## 변경 파일
- `scripts/daily.py` — 환율 레짐/FX 시그널/상관 리스크 터미널 표시 + JSON 출력
- `src/common.py` — consensus에 fx_signal 첨부

## 추가된 출력

### 터미널 (Rich)
1. 시장 환경 섹션: `💱 환율 레짐: 원화 약세 (MA20 > MA60)` + 극단 변동 경고
2. 포트폴리오 섹션: 상관 리스크 경고 (고상관 쌍, 섹터 집중, 분산도)
3. 다관점 판정 카드: FX 시그널 (BULLISH/BEARISH + 수출주/내수주 분류)

### JSON (--json)
- `fx_regime`: USD/KRW 환율 레짐
- `fx_regimes`: 다통화 레짐
- `portfolio.correlation_risk`: 고상관 쌍
- `portfolio.sector_concentration`: 섹터 집중도
- `portfolio.diversification_score`: 분산도 점수
- `multi_perspective[ticker].fx_signal`: 종목별 FX 시그널
